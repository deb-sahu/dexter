## My Pledge

I am committed to providing a welcoming and inspiring community for everyone. This Code of Conduct outlines my expectations for all those who participate in this community, as well as the consequences for unacceptable behavior.

## ü§ù Community Standards

### Expected Behavior

- **Be Respectful**: Treat everyone with respect. Healthy debate is encouraged, but always be courteous.
- **Be Collaborative**: Work together! This is a community project, and we're all here to make Dexter better.
- **Be Supportive**: Help others when you can. We all started somewhere.
- **Be Constructive**: When giving feedback, focus on being helpful and constructive.
- **Be Patient**: Remember that everyone has different skill levels and experiences.

### Unacceptable Behavior

- Harassment, discrimination, or offensive comments
- Trolling, insulting/derogatory comments, and personal attacks
- Public or private harassment
- Publishing others' private information without permission
- Spam or irrelevant promotions
- Any conduct which could reasonably be considered inappropriate

## üåü My Philosophy

### Why Open Source?

Dexter is open source because:
- **Transparency**: Users can verify security and privacy claims
- **Learning**: Developers can learn from real-world Flutter code
- **Collaboration**: The community can help improve the app
- **Trust**: Open code builds trust with users

### Why These Restrictions?

These guidelines exist to:
- Protect the integrity of the Dexter brand
- Prevent user confusion from unauthorized copies
- Ensure proper security and privacy standards
- Maintain a healthy, collaborative community

**I want contributors, not clones.**
